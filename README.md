

# SVG Frame Animator

![Python](https://img.shields.io/badge/Python-3.8%2B-blue?logo=python) ![PyQt5](https://img.shields.io/badge/PyQt5-Widgets-blue?logo=qt) ![License](https://img.shields.io/badge/License-MIT-green)

**SVG Frame Animator** is a lightweight Python tool for animating sequences of SVG files.

---

## Features

* Real-time folder watching: automatically updates when SVGs are added or removed
* Play / Pause / Step
* Adjustable FPS (1â€“120)
* Cross-platform (Windows, macOS, Linux)

---

## Installation and Usage

Clone the repository:

```bash
git clone https://github.com/CMaccapo/svg-frame-animatior.git
cd svg-frame-animatior
```

Install dependencies:

```bash
pip install PyQt5 watchdog
```

Run the app:

```bash
python -m src.main
```
